      1.1行内样式：写在标签的"style属性"中
      1.2内嵌样式：写在"style标签"中,通常约定写在head中
      1.3外联样式：写在一个单独的.css文件中，通过link标签引入

//3.基本选择器
3.1 通配选择器：*{ 

    }  可以选中所有HTML元素（标签）

3.2 元素选择器：h1{

}  选中特定的HTML中的元素（标签）

3.3 类选择器：.类名{

}  选中class属性值与该类名一致的元素。元素的class属性只能有一个，但class属性值可以有多个，用空格分隔，如class="c1 c2"。

//4.复合选择器:建立在基本选择器之上，由多个基础选择器，通过不同的方式组合而成
4.1 并集选择器：选择器1,选择器2,...,选择器n {
    
} 给多个选择器增加规则，用来设置多个选择器中重复的规则。

4.2 交集选择器：选择器1选择器2...选择器n {

} 选中同时符合多个选择器条件的元素。元素选择器出现且仅出现在最前面且只能有一个，后面的选择器可以有多个且顺序无关。
id选择器作为交集条件出现无意义。

4.3 后代选择器：父元素 子元素 {

} 选择某个元素的所有后代元素，无论层级如何。例如，div p 会选择所有在 div 内部的 p 元素，不论它们是直接子元素还是更深层的后代元素。

4.4 子代选择器：父元素>子元素 {

} 只选择某个元素的直接子元素。例如，div > p 只会选择那些直接在div下的p元素，而不会选择更深层的 p 元素。

4.5.1 兄弟选择器1：A + B{

} 选择紧接在A元素后面的第一个B元素。只有当B是A的直接后一个(紧挨着)兄弟元素时，B才会被选中。

4.5.2 兄弟选择器2：A ~ B{

} 选择所有与 A 同级的 B 元素，只要它们在 A 之后。

4.6.1 属性选择器1：[属性名] {

} 选中所有含有该属性的元素。

4.6.2 属性选择器2：[属性名="value"] {

} 选中所有含有该属性且值为value的元素。

4.6.3 属性选择器3：[属性名^="prefix"] {

} 选中所有含有该属性且值以"prefix"开头的元素。

4.6.4 属性选择器4：[属性名$="suffix"] {

} 选中所有含有该属性且值以"suffix"结尾的元素。

4.6.5 属性选择器5：[属性名*="content"] {

} 选中所有含有该属性且值含有"content"的元素。
